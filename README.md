# 泡泡鲸工具

一个基于Flask的Web应用程序，提供ASCII字符与各种进制之间的转换、位运算可视化、Shor算法整数分解以及SHA-256加密功能。

## 功能特点

1. **ASCII字符 → 进制转换**
   - 将单个ASCII字符转换为二进制、十进制和十六进制表示
   - 显示标准二进制和32位补零二进制表示

2. **进制数值 → ASCII字符**
   - 支持二进制、十进制和十六进制数值转换为对应的ASCII字符
   - 显示8位二进制表示，符合ASCII字节标准

3. **位运算可视化**
   - 支持与(&)、或(|)、异或(^)、非(~)、左移(<<)、右移(>>)位运算
   - 32位无符号整数运算限制
   - 直观的二进制位可视化展示

4. **Shor算法整数分解**
   - 递归分解整数得到所有质数因子
   - 显示因数分解结果和指数形式
   - 经典计算机上的Shor算法模拟实现

5. **SHA-256加密**
   - 将任意文本转换为SHA-256哈希值
   - 支持各种长度的文本输入
   - 符合NIST标准的SHA-256实现

## 技术栈

- **后端**: Flask 3.0.3
- **前端**: HTML5, Tailwind CSS v3, Font Awesome, JavaScript
- **部署**: 本地开发服务器或生产服务器

## 快速开始

### 安装依赖

```bash
pip install -r requirements.txt
```

### 运行应用

```bash
python app.py
```

然后在浏览器中访问 http://localhost:5000/ 即可使用应用。

## 开发说明

- 应用使用Flask作为Web框架
- 前端使用Tailwind CSS进行样式设计，实现了响应式布局
- 采用了AJAX技术实现无刷新交互
- 位运算功能严格遵循32位无符号整数运算规则

## 注意事项

1. ASCII转换功能仅支持单个字符输入
2. 进制转换时请确保输入格式正确
3. 位运算移位操作限制在0-31位范围内
4. 本应用仅支持标准ASCII字符（0-127）
