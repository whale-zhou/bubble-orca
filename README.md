# bubble-orca
"This is a tool that supports mutual conversion between number bases and ASCII, and visualizes them in 32-bit binary. Additionally, I have added the Shor algorithm and SHA-256 encryption algorithm that can be implemented online.这是一个支持进制与ASCII之间互相转换，并且用32位二进制可视化出来的工具。同时我新增了可在线实现的shor算法和SHA-256加密算法。"
