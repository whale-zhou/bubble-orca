关于 Shor 算法

Shor 算法是一种革命性的量子算法，由 Peter Shor 于 1994 年提出，专门用于高效分解大素数。该算法是第一个被证明能在量子计算机上以多项式时间解决 NP 问题的算法，对当前广泛使用的 RSA 加密体系构成了根本性威胁。

在经典计算机上，整数分解问题被认为是 NP 难的，没有已知的多项式时间算法。而 Shor 算法利用量子计算的叠加态和纠缠特性，特别是量子傅里叶变换，能够以 O((log N)^3) 的时间复杂度分解大整数，这是指数级的加速。

本实现是经典计算机上的优化模拟版本，完整展示了 Shor 量子算法的核心原理和工作流程，包括：

  • 量子叠加态的经典模拟
  • 量子傅里叶变换的原理展示
  • 周期寻找算法的优化实现
  • 完整的 Shor 算法工作流程

Shor 量子算法核心步骤

  1. 输入预处理：验证输入 N 是否为大于 1 的奇数，处理特殊情况
  2. 选择随机数：选择一个与 N 互质的随机数 a ∈ [2, N-1]
  3. 量子周期寻找：
    • 在量子计算机上，使用量子寄存器创建叠加态
    • 执行量子相位估计算法，利用量子傅里叶变换高效寻找周期 r
    • r 是满足 a^r ≡ 1 mod N 的最小正整数
  4. 经典后处理：
    • 如果 r 是偶数且 a^(r/2) ≢ -1 mod N，则计算可能的因子
    • 使用欧几里得算法计算 gcd(a^(r/2) ± 1, N)
    • 得到非平凡因子 p 和 q，使得 N = p × q
  5. 结果验证：检查得到的因子是否正确，否则重复步骤 2-4

量子优势的关键

量子傅里叶变换 (QFT) 是 Shor 算法实现量子优势的核心。在经典计算机上，傅里叶变换需要 O(n²) 时间，而量子傅里叶变换仅需要 O(n²) 个量子门操作，其中 n 是量子比特数。 更重要的是，量子傅里叶变换能够在单次测量中提取出周期信息，这是经典算法无法做到的。
